# Explore UI
<h3> Description :-</h3>
Explore UI Is a Light-weight and easy to use UI Component Library for developing fast and efficient web Interfaces.
<hr>
<h3>Technologies Used:-</h3>
<ul>
 <li>HTML</li>
 <li>CSS</li>
 </ul>
 <hr>
 <h3>Components Defined:-</h3>
 <ul>
 <li>Alert</li>
 <li>Avatar</li>
 <li>Badge</li>
 <li>Button</li>
 <li>Card</li>
 <li>Image</li>
 <li>Input</li>
 <li>Toast</li>
 <li>Navigation Bar</li>
 <li>List</li>
 </ul>
