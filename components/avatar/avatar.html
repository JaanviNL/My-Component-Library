Avatar

An avatar is the face of user's profile. It is the user profile picture. It is a small image consisting of user's face generally to give user's content a visual identity.

Round Avatar

There are multiple sizes for round avatars. These can be accessed by class names avatar-small , avatar-medium and avatar-large. Copy the code from below and add them accorging to your required size.

Square-avatar

Square shaped avatar can be accessed by class names- square-avatar-small , square-avatar-medium and square-avatar-large.
